1. Функция minify_cover содержит только заглушку pass
2. Ограниченный набор вспомогательных функций
3. Отсутствие type hints для функций
4. Константа strip_symbols используется только в fb2sax.py

1. Реализовать функцию minify_cover или удалить ее:

   - Если не нужна - удалить функцию и ее вызовы
2. Расширить набор утилитарных функций:
   - Функции для работы с путями и файлами
   - Функции для обработки текста

3. Добавить полную типизацию для всех функций
4. Оптимизировать функцию normalize_string с кэшированием regex
5. Реорганизовать модуль для лучшей структуры:
   - Либо разделить на подмодули
   - Либо сгруппировать функции по назначению
6. Добавить тесты для всех функций
7. Удалить неиспользуемую константу strip_symbols или переместить в fb2sax.py
