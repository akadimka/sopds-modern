3. Отсутствие проверки входных данных в методах

1. Исправить метод mime_by_type:

   - Добавить проверку на существование ключа
2. Сделать словарь suffix_mime_mapper приватным или неизменяемым
3. Добавить валидацию входных параметров в методах
4. Добавить метод для получения расширения по MIME-типу
5. Добавить поддержку дополнительных форматов книг
