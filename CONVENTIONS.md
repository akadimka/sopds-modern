## Краткие конвенции проекта

- **Python:** 3.10+, управление зависимостями через `uv`, виртуальное окружение `.venv`.
- **Форматирование:** `ruff`, отступы 4 пробела, длина строки до 88 символов.
- **Импорты:** Группировка (стандартные → сторонние → локальные), каждый блок отделён пустой строкой. Без `import *`.
- **Структура:** Исходный код в `src/`, тесты в `tests/`, фикстуры в `tests/fixtures/` (реэкспорт через `__init__.py`).
- **Именование:**
  - `snake_case` – переменные, функции.
  - `PascalCase` – классы.
  - `UPPER_SNAKE_CASE` – константы.
- **Типизация:** Аннотации типов для всех публичных функций/методов.
- **Документирование:** Docstrings в стиле **ReST** на русском языке для публичных API.
- **Логирование:** Только `logging` (иерархические логгеры через `__name__`), без `print()`.
- **Обработка ошибок:** Конкретные исключения в `except`, обязательная привязка `as`, логирование/проброс ошибки, никаких `pass`.
- **Тестирование:** `pytest`, параметризация через `pytest.mark.parametrize`, фикстуры сгруппированы по модулям.
- **Работа с AI:** Читать предоставленные файлы, изменять только указанные, не добавлять служебных комментариев, проверять тесты.
